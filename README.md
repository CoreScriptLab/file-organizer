File Organizer (Free Version)

A simple Python tool that automatically organizes files in a folder by their extension.
Part of CoreScriptLab.

Features (Free Version)

Organizes files by extension

Creates folders automatically

Ultra lightweight

No configuration needed

How to Use

Install Python 3

Download file_organizer_basic.py

Run it:

python file_organizer_basic.py


Enter the folder path

Done
